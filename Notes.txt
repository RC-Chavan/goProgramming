Memory Management :-
In go lang memeory management in not your job, it will handled by go internally.

Memory Management
2 ways :- 
1. New() - 
    In new Memory is allocated but no initalized
    You will get a memory address
    Zero is stored
2. make()
    In make Memory is allocated and aslo initalized
    here also you will get memory address
    non-zeroed storage

majority cases we will use make in go


garbage collection happens automatically

out of scope or nil
